# 8th Light Accessibility Community of Practice

This repository holds the charter, decisions, and workflows for 8th Light's Accessibility Community of Practice.

## What Lives Here

- **CoP Charter** - Our purpose, operating model, and structure
- **Decision Records** - Key decisions made during working sessions
- **AI Workflows** - LLM-assisted accessibility review workflows (coming soon)
- **Artifacts from Sessions** - Templates, runbooks, and tools created during working sessions

## What Lives Elsewhere

- **Accessibility Playbook (Google Docs)** - General guidance, training materials, and reference documentation
- **Project work** - Client-specific accessibility work stays in project repos

## How to Use This Repo

- **Read the Charter first:** [charter/8L-A11y-CoP-Charter.md](charter/8L-A11y-CoP-Charter.md)
- **Propose changes:** Open an issue or PR
- **Add artifacts:** Most artifacts are created during working sessions, not in advance

### For CoP Scribes

After a working session, follow the step-by-step workflow documented in [docs/SCRIBE-WORKFLOW.md](docs/SCRIBE-WORKFLOW.md).

**Optional:** Use `/scribe-workflow` in Claude Code for guided assistance. The skill walks you through the workflow and calls the necessary skills when those steps are reached (e.g., `/scribe-retrospective`, `/workflow-feedback`). If following the manual workflow, you can invoke these skills yourself at the appropriate steps.

The documentation is the authoritative source. The skill is a convenience wrapper that follows the same workflow.
